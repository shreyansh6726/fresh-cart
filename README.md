# Fresh-Cart E-commerce Mockup

Fresh-Cart is a minimalist, responsive front-end mockup for an online grocery or fresh produce e-commerce platform. The focus is on a clean, modern user experience using a vibrant color palette and direct navigation.

##  Features

* **Responsive Header:** Uses a two-row layout to separate the centralized brand logo from the navigation links.
* **Intuitive Navigation:** Includes multi-level dropdown menus for key sections like **Categories**, **About Us**, **Feedback**, and **Profile**.
* **Quick Quantity Control:** Allows users to increment or decrement product quantity directly on the main product cards.
* **Dynamic Cart Sidebar:** A non-intrusive sidebar displays cart items, quantities, and the running total.
* **Product Filtering:** Includes a search bar ready for back-end integration to filter displayed products.
* **Custom Color Palette:** Utilizes a custom green and warm accent color scheme for a fresh aesthetic.

##  Technology Stack

| File | Purpose | Key Technologies |
| :--- | :--- | :--- |
| `index.html` | Project Structure | HTML5, Semantic Elements |
| `style.css` | Styling and Layout | CSS3, Flexbox, CSS Variables, Media Queries |
| `script.js` | Interactivity | Vanilla JavaScript, DOM Manipulation |

##  How to Run

1.  Clone or download the project files.
2.  Ensure you have `index.html`, `style.css`, `script.js`, and placeholder images (`logo.png`, `placeholder-apple.jpg`, etc.) in the same directory.
3.  Open `index.html` in any modern web browser (Chrome, Firefox, Edge).
